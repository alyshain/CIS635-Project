Already done:

The fusion performed in this project is Early Fusion (also called Data-Level Fusion).

Combined three heterogeneous data sources — SEDA achievement data, F-33 finance data, and SEDA covariate/demographic data — into a single flat merged dataset before any analysis was run.

All variables from all sources were joined together at the district level using leaid as the shared key, producing one unified dataframe that was then fed into the correlation and regression analyses.

Per Meng et al., early fusion combines raw data from multiple sources into a single unified representation before any learning or analysis occurs. This is already done - joining SEDA, F-33, and the covariate file at the leaid level into one flat dataframe before running any models.
The advantage Meng et al. note for this approach is that it preserves inter-source correlations (heatmap captures relationships like LocalRevPP correlating with both SES_Composite and Achievement simultaneously).

Miscellaneous Notes for reference
Core idea: “Does per-pupil school funding predict student academic achievement, and what role does neighborhood wealth play?”
Dataset 1 — Student Academic Performance
Source: Stanford Educational Opportunity Project - 2009-2019
Link: https://edopportunity.org/opportunity/

The Stanford Education Data Archive (SEDA) includes a number of publicly available data files, the technical documentation and data codebooks, listed on the following page. Data files are available in Stata and .csv formats.

In publications, please cite the data as:
Reardon, S. F., Ho, A. D., Shear, B. R., Fahle, E. M., Kalogrides, D., saliba, j. (2026).
Stanford Education Data Archive (Version 6.0). Retrieved from https://purl.stanford.edu/xh833nn4025.

The currently available data include average achievement (for all students and by race/ethnicity and gender), racial/ethnic and gender achievement gaps, and demographic/socioeconomic data. We report multiple aggregations of the data: school, geographic district, administrative district, county, metropolitan statistical area, commuting zone, and state.

The most recent release (currently, Version 6.0) should always be used for reporting and analysis. Previous versions of the data are still available to facilitate research replication. Please review the technical documentation and codebooks that accompany the data sets. These documents review the data construction process and describe the contents of each file.

Achievement Score Columns
The naming follows a pattern: cs = Cohort Scale, mn = mean, then the measure, then ol or eb, then optionally se or se_adj
Suffix
Meaning
_ol
OLS (ordinary least squares) estimate
_eb
Empirical Bayes estimate (shrinkage, more reliable for small districts)
_se
Standard error of the estimate
_se_adj
Adjusted standard error accounting for sampling error



Measure
Meaning

cs_mn_tlr
Trend in learning rate over time

Mean achievement in math only

So cs_mn_avg_eb = cohort scale mean average achievement, Empirical Bayes estimate — this is your primary outcome variable.
